Description

该包下的类是对 flagwind map 基础设计。主要设计原则如下：

  • 剔除地图相关的特有概念，设计统一模型、接口。
  • 与地图相关操作抽象成abstract方法供不同地图api实现。
  • 非地图相关操作设计成基础方法供子类调用。
